Bungee View text search syntax

Entering a set of keywords matches items that contain any of those keywords, in any of the item's metadata. Text searches are not case-sensitive; in other words, capitalization is ignored.

To search for items containing all words, put '+' in front of each word (or enter each word one at a time and type enter).

To search for items not containing a word, put '-' in front of the word. (You must have at least one word that doesn't have a '-' in front of it, though.)

You can add a wildcard at the end of a sequence of letters. For example, 'bun*' will match 'bun', 'buns', or 'bungee'.

You can search for a phrase by putting it in quotation marks, for example "grand canyon".

For more details, see the complete documentation for MySQL Boolean Mode searches.
