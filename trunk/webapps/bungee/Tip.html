Bungee View Tips and Tricks

Notice the Mouse Documentation Line at the bottom of the header
As you move the mouse, it tells you what will happen if you click. In Expert Mode if the mouse is not over anything you can click on, it tells you what the arrow keys will do.

How to determine what filters are in effect
There is an English-like description of your current query above the buttons in the header. If it doesn't fit on one line, you can move the mouse over it and see the full description. Each filter term is also highlighted in bright green, both along the left edge of the Tag Wall column, as well as when it appears in the Selected Result column. Since the selected result satisfies all the filters, all filter terms will all appear there (unless in Expert Mode you have selected multiple tags in one category, or selected a tag's brown x checkbox).

Navigate in the Selected Result column
You can add and remove filters by clicking on tags for the selected result just like you can on the bars and labels on the Tag Wall. Since you can see all the categories and tags at once, this is faster for the common case where the selected result has the tags you are interested in.

Navigate in the Results column (Expert Mode only)
After you click on a thumbnail in the Results column, you can use the arrow keys to select adjacent thumbnails.

Navigate in the Tag Wall (Expert Mode only)
After you click on a bar or label on the Tag Wall, you can use the right and left arrow keys to select adjacent tags. This is especially useful when there are too many tags to fit on the screen. You can get a list of all tags in the selected category by clicking on the category name.

Filter on multiple tags for a category (Expert Mode only)
Use control-click to add (or remove) additional single tags, or shift-click to add a range of tags to a filter on a category. The filter will match items that have at least one of the selected tags for that category. Clicking on the light green checkbox in front of a tag label has the same effect as control-clicking. Clicking on the light brown checkbox filters out items with that tag.

Control how your browser shows help pages and result details
Bungee View tells your default browser what web page to display, but the browser decides whether to replace the current page, open a new window, or open a new tab. Some browsers let you customize this behavior. For instance, in Firefox go to Tools>Options>Tabs.

Layout Customization (Expert Mode only)
The initial text size is set to match that of the browser that launched Bungee View. You can change it interactively by dragging the header border up and down. Move the mouse over the border until you see a horizontal white line, and then drag the line. You can also adjust the widths of the Tag Wall, Matches, and Selected Result columns by dragging their borders. (If you resize the whole window, it will compute new column widths and lose your changes.) To change the number of columns of thumbnails, move the mouse over them until you see a white grid, and then drag the grid.

Get rid of those annoying pop-ups
Typing control+P (for Pop-up) toggles between showing tool tips on the Tag Wall and in the mouse documentation line in the header.

Find the most relevant tags to your query
Typing control+C (for Cluster) lists the tags most closely associated with your current query, in order of relevance. This is more precise than looking for tags colored dull green.
