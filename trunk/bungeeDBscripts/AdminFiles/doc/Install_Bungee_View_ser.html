1. Install Tomcat



2. Move ../../bungee/ to $CATALINA_HOME/webapps/


$CATALINA_HOME is the directory where you installed Tomcat, for instance C:\Program Files\Apache Software Foundation\Tomcat 5.5\

The hierarchy should look like this:

$CATALINA_HOME/webapps/bungee

    Abo.html
    Edi.jpg
    Sea.html
    TagWallExplained.gif
    Tip.html
    Wha.html
    bungeeClient.jar
    bungee.gif
    bungee.jsp
    downloadJavaWS.html
    gpl.html
    gpl_files
    index.html
    missing.gif

    WEB-INF

        web.xml

        lib

            bungeeServlet.jar

            mysql-connector-java-5.0.4-bin.jar

            pjl-comp-filter-1.6.6.jar



3. Edit web.xml


A. Change p5pass to the password you assigned to the bungee MySQL user.


B. Remove the names of all the databases except wpa.


C. If Tomcat is running on a different computer from MySQL, also change "localhost" to the name of the database server.


4. Manage Tomcat


You can start and stop the servlet from http://localhost/manager/html

You should do this any time you change web.xml (i.e. do it now) or any of the jar files in WEB-INF\lib.

You can run Bungee View from http://localhost/bungee/.

If there is a problem, look in $CATALINA_HOME/logs/ for the latest log files.
