2. Move ../../wpa/ to the MySQL data directory (e.g. C:\Program Files\MySQL\MySQL Server 5.1\data\) and restart MySQL.

wpa is the WPA Posters collection from the Library of Congress American Memory Collection.

3. Create user bungee and assign permissions

Substitute your own password for PASSWORD
C:\Program Files\MySQL\MySQL Server 5.0\bin>mysql -u root -p
Enter password: ********
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 15
Server version: 5.0.41-community-nt-log MySQL Community Edition (GPL)

Type 'help;' or '\h' for help. Type '\c' to clear the buffer.

mysql> CREATE USER bungee@localhost IDENTIFIED BY 'PASSWORD';

mysql> GRANT SELECT, INSERT, UPDATE, CREATE, DELETE, ALTER ROUTINE, EXECUTE, CREATE VIEW, CREATE TEMPORARY TABLES, DROP ON wpa.* TO bungee@localhost;

C:\Program Files\MySQL\MySQL Server 5.0\bin>exit
