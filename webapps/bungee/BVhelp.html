What to do if Bungee View Doesn't Start

  • If nothing happens, your browser may have downloaded the Java Web Start file without opening it. Show recent downloads (on Safari, for instance, click on the down arrow icon), and click on the Web Start file (bungee.jnlp or similar) to start Bungee View.
  • If your browser doesn't know how to open the link, it means you need to install Java version 7 or later.
  • On Mac OS, Gatekeeper may be preventing you from opening an App that isn't from the Apple Store or signed with an Apple Developer ID. Java has its own code-signing security, and you can either enable the Bungee View App individually, or set Gatekeeper to allow all non-Apple Apps to run. See the Apple Gatekeeper instructions.
  • If you still have problems, let me know and I will try my best to help you.
